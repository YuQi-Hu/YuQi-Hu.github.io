do key<-a[j]
i<-j-1
while i>0 and a[i]>key
do a[i+1]<-A[i]
i<-i-1
a[i+1]<-key
for(int j=1;j<n;j++)
{
key=a[j];
for(int i=j-1;i>=0;)
{

}

定义与赋值

指针数组

指针数组与二维数组的区别

二维数组 所有元素按行一次连续存放
指针数组 每个指针不一定连续存放

以指针作为函数参数

值传递/引用传递（双向传递）
传递一组数据时，使用指针只传首地址

读入三个浮点数，将整数部分和小数部分分别输出
